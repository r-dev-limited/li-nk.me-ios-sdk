LinkMe iOS SDK

Universal links, deferred deep linking, and attribution for iOS apps. Route users to the right screen — even on first install.

Setup Guide CocoaPods

1 Install

Swift Package Manager:

https://github.com/r-dev-limited/li-nk.me-ios-sdk

CocoaPods:

pod 'LinkMeKit', '~> 0.2.13'

2 Initialize

import LinkMeKit

LinkMe.shared.configure(config: .init(
    appId: "your_app_id",
    appKey: "your_app_key",
    sendDeviceInfo: true,
    includeVendorId: true,
    debug: true
))

3 Handle Links

// Cold-start link
LinkMe.shared.getInitialLink { initial in
    if let initial {
        routeUser(to: initial)
    } else {
        // Deferred deep link (first install)
        LinkMe.shared.claimDeferredIfAvailable { deferred in
            if let deferred { routeUser(to: deferred) }
        }
    }
}

// Live links while app is running
let unsubscribe = LinkMe.shared.addListener { payload in
    routeUser(to: payload)
}

Features

Universal Links

HTTPS deep links via Apple Universal Links. LinkMe hosts the AASA file automatically for your domains.

Deferred Deep Linking

Route users on first install via pasteboard tokens (deterministic) or fingerprint matching (fallback).

Analytics

Track app events and attribute installs to campaigns with track() and UTM parameters.

SwiftUI & UIKit

Works with onOpenURL, onContinueUserActivity, and AppDelegate patterns.
