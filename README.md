## Installation

### Swift Package Manager
Add `https://github.com/r-dev-limited/li-nk.me-ios-sdk` to your package dependencies and select version `0.1.2` (tag `v0.1.2`).

### CocoaPods
```ruby
pod 'LinkMeKit', '~> 0.1.2'
```
`LinkMeKit 0.1.2` is available on CocoaPods trunk ([spec commit](https://github.com/CocoaPods/Specs/commit/5c66ac598927c91a445652e1419a232325ed2899)).
